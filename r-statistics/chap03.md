## 3.3 共分散

共分散は偏差の積の平均

``` r
共分散1と2 <- mean((統計テスト1 - mean(統計テスト1)) * (統計テスト2 - mean(統計テスト2)))
共分散1と2
```

    ## [1] 7.55

不偏共分散

``` r
cov(統計テスト1, 統計テスト2)
```

    ## [1] 7.947368

## 3.4 相関係数

``` r
cov(統計テスト1, 統計テスト2) / (sd(統計テスト1) * sd(統計テスト2))
```
